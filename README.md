
# AwsStorageApi

This project is to save, read and delete files on AWS S3.


## Documentation

Update parameters in appsettings.json.

```json
{
  "AwsConfiguration": {
    "AccessKey": <AccessKey AWS>,
    "SecretKey":  <SecretKey AWS>,
    "BucketName":  <BucketName AWS>
  }
}
````

Run project and test on Swagger.
